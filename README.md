# JovioTrendz - Luxury Landing Page

## ✨ Overview

A stunning dark, cinematic luxury landing page built with React, TypeScript, and TanStack Start. Features handcrafted animations, premium design elements, and seamless user experience.

## 🎨 Design Features

- **Dark Cinematic Style** - Matte black (#0f172a) with gold accents (#d97706)
- **Serif Typography** - Cormorant Garamond for premium feel
- **Smooth Animations** - Framer Motion scroll-reveal effects
- **Responsive Design** - Mobile, tablet, and desktop optimized
- **Premium Components**:
  - Hero section with cinematic intro
  - Limited edition showcase (3 curated items)
  - Trust strip with credibility metrics
  - Elite Club membership section
  - Professional footer with social links

## 🛠️ Tech Stack

- **Framework**: React 19 + TypeScript
- **Build**: Vite + TanStack Start
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **UI Components**: Lucide React Icons
- **Forms**: React Hook Form + Zod
- **UI Library**: Radix UI

## 🚀 Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Production Build

```bash
npm run build
```

```bash
npm run preview
```
